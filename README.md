# 介绍


`jk-ui` 是一款基于 `Vue.js 2.0` 的前端 UI 组件库, 其中文件夹package为jk-ui源码
整个项目为jk-ui的API

## 特性

- 基于 `Vue` 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES6
- 提供友好的 API，可灵活的使用组件

## 启动
>1. npm intall
>2. npm run dev

## 使用jk-ui
1. npm install --save jk-ui
2. import JKUI from 'jk-ui'
3. import 'jk-ui/theme-default/lib/index.css'
4. Vue.use(JKUI)
